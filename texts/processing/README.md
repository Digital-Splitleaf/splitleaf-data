# Texts In Process
This folder, and all of its subfolders, contains **texts in progress** generated automatically by our algorithm. Because these files are not yet finalized, **community review and contributions are essential**. Please follow the guidelines below to help ensure accuracy, consistency, and TEI compliance.

---

## ✅ Purpose of This Folder

The materials here are **draft versions** of TEI‑encoded texts. They may include placeholder values, automatically generated syllabification, and other content that still requires correction or expansion.

Your contributions will help move these files toward a fully reliable, publication‑ready state.

---

## 🛠️ Community Editing Priorities

### **1. Replace placeholder text with correct information**
Many files contain placeholders such as: AUTHOR_HERE

Please replace these with accurate information based on the text you are editing.

---

### **2. Confirm the correctness of syllable divisions**
All syllable boundaries in these files were produced automatically and **must be manually verified**.

⚠️ **Important:**  
Syllables in *musical lyrics* **are not broken according to standard dictionary hyphenation**.  
Please ensure that syllable divisions reflect *lyrical and musical practice*, adjusting them wherever needed.

---

### **3. Review and correct all remaining content**
Verify the accuracy of:

- Spelling and transcription  
- TEI structure  
- Metadata  
- Punctuation and formatting  
- Logical structure and completeness  
- Removal of any remaining placeholders

---

### **4. Improve the `<teiHeader>` following TEI guidelines**
You may add any additional information to the `teiHeader` that enhances clarity, documentation, or metadata quality.

Please ensure all edits conform to the official TEI P5 guidelines:

📘 **TEI Documentation:**  
https://tei-c.org/release/doc/tei-p5-doc/en/html/index.html

As you edit, be sure to:

- Use only valid TEI elements and attributes  
- Follow correct hierarchical structure  
- Avoid introducing non‑standard constructs  
- Add comments if something requires later review

---

## 🤝 Contribution Workflow

1. Select a file you wish to improve  
2. Make edits based on the priorities listed above  
3. Commit your changes with a clear description  
4. If the file needs further review, open a Pull Request  
5. Tag maintainers or other contributors as needed  

---

## Thank You

Your contributions, large or small, make this project stronger, more accurate, and more useful to the wider community.  
Thank you for helping build a cleaner, more reliable corpus of metrical psalm texts!
